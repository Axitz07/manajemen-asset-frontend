# Manajemen Asset

Starter project untuk aplikasi manajemen aset berbasis Vue 3 + Vite.

## Struktur awal

```text
src/
|- assets/
|  |- styles/
|- components/
|  |- common/
|  |- dashboard/
|- composables/
|- constants/
|- data/
|- layouts/
|- services/
|- stores/
|- utils/
|- views/
```

## Menjalankan project

```bash
npm run dev
```

## Catatan

Struktur ini masih fondasi awal. Selanjutnya kita bisa lanjut membuat:

- router halaman
- data dummy aset
- dashboard ringkasan
- form input aset
- tabel daftar aset
